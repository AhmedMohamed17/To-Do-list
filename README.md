# To-Do-list
a simple to do list app to manage your tasks.

## Languages and tools :

React.js - Hooks 

## Requirements

For development, you will only need Node.js installed on your environement.

## Install

   $ git clone https://github.com/AhmedMohamed17/To-Do-list.git
   $ cd To-Do-list
   $ npm install
   
   ## Start & watch

   $ npm start
